Who is Web Developer?

A web developer or programmer is someone who takes a web design - which has been created by either a client or a design team - and turns into a website

They do this by writing lines and lines of complicated code, using a variety of language. Webdevelopers have quite a difficult job, because they essentially have to take a language we understand, such as Chinese English, and translate it into a language that a computer understands, such as Python ot HTMl.

As you can imagine, this can take a lot of time and effort and requires an intricate understanding of various programming languages and how they are used. Different types of developers specialize in different areas, which means that large web projects are usually a collaboration between several different developers.

Types of Developers

There are three main type of developers such as :

Front-End developers
Back-End Developers
